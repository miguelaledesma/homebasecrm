# Logo Files

Please add your logo files to this directory:

1. **a1-logo.png** - A1 Interior & Exterior Designing logo
   - Recommended size: 240x80px or larger (will be scaled down)
   - Format: PNG with transparent background preferred

2. **ez-floors-logo.png** - EZ FLOORS & MORE logo  
   - Recommended size: 100x100px or larger (circular logo)
   - Format: PNG with transparent background preferred

The logos will be displayed in:
- Header/navigation bar (both logos side by side)
- Sign-in page (both logos at the top)

If you have SVG versions, you can use those instead - just update the file extensions in:
- `components/logo.tsx`
- `app/auth/signin/page.tsx`
